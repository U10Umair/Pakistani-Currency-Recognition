## Problem
The goal of this project is to identify the denomination of Pakistani currency notes using image classification techniques.

## Approach
Each currency note denomination is treated as a separate class. The model will learn visual patterns from labeled images to predict the correct note value.

## Dataset
The dataset consists of images of Pakistani currency notes organized by denomination.
